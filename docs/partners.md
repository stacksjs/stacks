# Partners

Stacks works with maintainers, teams, and ecosystem projects that improve TypeScript and Bun development.

For partnership inquiries, open a discussion in the [Stacks repository](https://github.com/stacksjs/stacks/discussions) with the proposed collaboration, expected user benefit, and any technical integration details.
